# devour-site

[devour.jp](https://devour.jp/) の静的サイトソースです。GitHub Pages でホスティングしています。

## サイト構成

| パス | 内容 |
|------|------|
| `/` | トップページ |
| `/apps/devour-browser-tools/` | devour browser tools（Chrome 拡張機能） |
| `/privacy` | プライバシーポリシー |
| `/support` | サポート・お問い合わせ |

## 技術スタック

- 静的 HTML / CSS（フレームワーク不使用）
- GitHub Pages + カスタムドメイン（devour.jp）

## ローカルプレビュー

任意の静的ファイルサーバーで確認できます。

```bash
python3 -m http.server 8000
```
