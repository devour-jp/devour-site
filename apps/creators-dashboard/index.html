Creators Dashboard（仮）（beta）

Openv1.3.0

つくる人の、いま。これから。

名前は仮称、現在ベータ版で PoC（実証実験）中です。 詳しくは紹介記事（note）・紹介記事（Zenn）をご覧ください。

A card-based dashboard that aggregates indie dev project info. Add a topic to your GitHub repo and get listed automatically.

概要

個人開発プロジェクトのリリース情報を集約するカード型ダッシュボードです。 GitHub リポジトリにトピックを追加するだけでプロジェクトがカード形式で自動掲載されます。

スクリーンショット

特徴

  • GitHub リポジトリにトピックを追加するだけで掲載（1分で試せる）
  • 最新バージョン・更新日・リリース内容をカードに表示
  • 次のバージョン予告で「これからの計画」を共有
  • 作者のひとことコメント（巻末コメント風）
  • App Store / Google Play バッジ対応
  • catalog-info.yaml で詳細情報をカスタマイズ可能

使い方

  1. GitHub リポジトリに creators-dashboard トピックを追加
  2. 自動的にカードが生成されて掲載
  3. より詳しい情報を載せたい場合は catalog-info.yaml をリポジトリルートに配置
