devour browser tools

デスクトップ版 Chrome 専用の拡張機能です。

概要

Web ブラウジングを快適にするツールセットです。

目的

  1. ドメインごとにコピー対象を設定でき、右クリックメニューからコピー対象のテキストを取得
    → 特定サイトでの繰り返しコピー作業を効率化する
  2. Zenn の投稿・編集 UI を無効化（CLI 運用サポート）
    → CLI/Git での記事管理を徹底し、誤操作を防止する

機能

1. Per-domain Copy

ドメインごとに CSS Selector を登録し、右クリックメニューからコピー対象のテキストを取得できます。

  • ドメイン + CSS Selector のルールをオプション画面で管理
  • サブドメインの自動マッチングに対応（例: example.com は docs.example.com にもマッチ）
  • 右クリック → "Copy preset element (domain rule)" で実行

Copy/Paste Toggle

拡張アイコンのクリックで、2ステップのコピー＆ペースト操作を実現します。iframe をまたいだテキスト操作に便利です。

  • 1回目のクリック: 指定要素のテキストをコピー（バッジに「P」表示）
  • 2回目のクリック: フォーカス中の入力欄にペーストを実行
  • 対象要素は popup から ID 指定が可能

2. Zenn Web Editor ブロック

Zenn の Web エディタ機能を無効化し、CLI での記事管理を徹底するためのツールです。

  • 投稿・編集ボタンをグレーアウト＋ツールチップで通知
  • 編集ページへのアクセスを自動リダイレクト

動作環境

  • デスクトップ版 Google Chrome
  • Manifest V3

モバイル版 Chrome には対応していません。
