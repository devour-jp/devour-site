devour browser tools — Demo

このページでは、拡張機能の主要機能を実際に試すことができます。3つのコピー方法と、ペーストトグルを確認できます。

前提

  • この拡張機能には devour.jp + #content がビルトインルールとして登録されています
  • 全ての操作はブラウザ内で完結し、外部への通信は発生しません

方法 A: 拡張アイコンでコピー

ツールバーの拡張アイコンをクリックしてコピーする方法です。

  1. ツールバーの拡張アイコンをクリックします
  2. options で設定した Target element ID（またはページ全体）のテキストがコピーされます
  3. 拡張アイコンに "P" バッジが表示されます（ペースト待機状態）

方法 B: 右クリック → Copy preset element

ドメインルールに基づいてコピーする方法です。

  1. このページ上の任意の場所で右クリックします
  2. コンテキストメニューから "Copy preset element (domain rule)" を選択します
  3. ビルトインルール（devour.jp → #content）に基づき、このページの <main id="content"> のテキストがコピーされます
  4. 拡張アイコンに "P" バッジが表示されます

方法 C: 右クリック → Copy element innerText

右クリックした要素のテキストを直接コピーする方法です。

  1. コピーしたい要素（例: この段落）を右クリックします
  2. コンテキストメニューから "Copy element innerText" を選択します
  3. 右クリックした要素の innerText がコピーされます
  4. 拡張アイコンに "P" バッジが表示されます

ペースト（共通）

上記 A / B / C いずれかでコピーした後、以下の手順でペーストします。

  1. 下のテキストエリアをクリックしてフォーカスします
  2. ツールバーの拡張アイコンをクリックします
  3. コピーしたテキストがテキストエリアにペーストされます
  4. "P" バッジが消え、待機状態に戻ります

テスト用テキストエリア

ペースト先として使用してください。

確認ポイント

  • 方法 A / B / C それぞれでコピーが成功すること
  • コピー後、"P" バッジが表示されペースト待機状態になること
  • テキストエリアにフォーカス後、アイコンクリックでペーストされること
  • ペースト完了後、"P" バッジが消えて待機状態に戻ること
